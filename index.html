How to Embed This Widget on Your Site

The error Cannot GET /index.tsx happens because web servers are not configured to send raw .tsx source code. You must first build or compile your project into a standard JavaScript file (e.g., widget.js) that browsers can run.

  1. Build Your Project: Use a build tool like Vite or Create React App. Run the build command (e.g., npm run build). This will create a production-ready folder, usually named dist.
  2. Deploy the Build Output: Upload the contents of the dist folder to your hosting provider (like Netlify, Vercel, or Google Cloud Run).
  3. Use the Correct Script Tag: Copy the code below and paste it before the closing </body> tag on your page.

Important: Replace https://your-chatbot-app-domain.com/assets/widget.js with the actual URL to the compiled JavaScript file from your deployment. 

<script type="module" src="https://your-chatbot-app-domain.com/assets/widget.js"></script>

Note: Your server hosting the chatbot must be configured with CORS (Cross-Origin Resource Sharing) headers to allow other domains to load this script.
